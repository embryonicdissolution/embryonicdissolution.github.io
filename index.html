Today I learned about additional CSS text properties.
For example, how to adjust line height, font styles, text alignment, font size, and more.
There is quite a bit to remember, but that is going to be the most difficult aspect of the process.

The application of these newly learned properties seems to be going fairly smoothly though; despite having to reference notes.
Overall, I'm pleased with my progress thus far.

Learned Today

Property Meaning
font-family:; A font family is a collection of fonts that share particular design features within a specific style of typeface.
font-size:; Determines font size
font-weight:bold; Is a way to change text to bold without abusing strong tags
font-style:italic; Is a way to italicize text without abusing em tags.
line-height:; To change spacing between lines in a paragraph
text-align Is used to specify the horizontal alignment of text in an element.
text-decoration:; Sets the appearance of decorative lines on text. Can use to underline, or remove underlining.
text-shadow:; Text shadow refers to adding layers to the text, i.e., giving the text a shadow whether to enhance, highlight, or increase the visual appeal of specific text by giving the text depth.
